# Pichip - K线形态相似度匹配工具

精确匹配全A股历史K线形态，发现投资机会。

## 功能特性

- **形态匹配**：在历史数据中寻找与目标走势相似的K线形态
- **量能分析**：支持换手率、量比相似度计算
- **形态识别**：支持多种K线形态自动识别
  - **首板二波**：涨停后缩量调整再启动的形态
  - **强势二波**：大涨后震荡整理再启动的龙头股二波机会
  - **涨停反弹二波**：涨停后回调至支撑位企稳反弹的形态
  - **揉搓线**：长上影+长下影组合，主力洗盘后拉升信号
- **主力控盘指数**：量化主力资金控盘程度
  - 筹码集中度：股东户数变化分析
  - 量价控盘：缩量上涨占比分析
  - 抗跌性：大盘下跌日个股表现分析
- **形态回归分析**：统计历史匹配的后续走势和胜率
- **定时任务**：自动同步数据、验证历史匹配
- **自然语言交互**：通过对话方式执行操作

## 安装

```bash
# 克隆项目
git clone https://github.com/your-repo/pichip.git
cd pichip

# 安装依赖
pip install -e .

# 或使用 pip3
pip3 install -e .
```

## 快速开始

### 1. 同步数据

首次使用需要同步A股历史数据：

```bash
# 同步最近3年数据（默认）
pichip sync

# 仅同步今日数据（收盘后使用，速度快）
pichip sync --today

# 中午同步（同步盘中实时数据，用于中午复盘）
pichip sync --intraday

# 中午同步指定股票
pichip sync --intraday --stock 603912
pichip sync --intraday --stock 603912,002837

# 指定时间范围
pichip sync --start-date 20230101 --end-date 20240222
```

### 2. 形态匹配

查找与目标股票走势相似的历史形态：

```bash
# 基本用法
pichip match 002594 2024-01-01 2024-02-01

# 匹配最新形态（推荐用于发现当前机会）
pichip match 002594 2024-01-01 2024-02-01 --latest

# 加入量能相似度
pichip match 002594 2024-01-01 2024-02-01 --latest --volume-weight 0.3

# 限定板块/概念
pichip match 002594 2024-01-01 2024-02-01 --board "汽车整车" --concept "新能源"

# 限定市值范围
pichip match 002594 2024-01-01 2024-02-01 --min-mv 50 --max-mv 500

# 生成对比图
pichip match 002594 2024-01-01 2024-02-01 --chart
```

### 3. 形态识别

扫描市场中的各种K线形态：

```bash
# 扫描全部形态（推荐）
pichip pattern --type all

# 首板二波（默认）
pichip pattern
pichip pattern --stock 002594

# 强势二波
pichip pattern --type strong_second_wave

# 涨停反弹二波
pichip pattern --type rebound_second_wave

# 揉搓线
pichip pattern --type rubbing_line

# 生成K线图
pichip pattern --type rubbing_line --chart
```

**支持的形态类型**：

| 类型 | 说明 |
|------|------|
| `all` | 全部形态：一次性扫描所有形态 |
| `first_board_second_wave` | 首板二波：涨停后缩量调整再启动 |
| `strong_second_wave` | 强势二波：大涨后震荡整理再启动 |
| `rebound_second_wave` | 涨停反弹二波：涨停后回调至支撑位企稳反弹 |
| `rubbing_line` | 揉搓线：长上影+长下影组合洗盘信号 |

#### 首板二波

识别连板后断板、缩量两小阳买入信号：

**识别条件**：
- **连板涨停**：至少1个涨停（首板或连板均可）
- **断板调整**：涨停后断板（当天不涨停）
- **支撑不破**：断板后回调不跌破断板当天的最低价
- **缩量两阳**：出现两根缩量小阳线（核心买入信号）

**核心逻辑**：
- 缩量 = 抛压小、分歧小、主力锁仓
- 小阳 = 不是放量大阳（放量大阳反而是离场信号）
- 小阳含十字星级别（涨幅接近0也算）
- 两阳不要求严格连续，中间可穿插1-2天小阴线或十字星
- 时间不限 = 断板后无论多久，只要出现缩量两小阳就是有效信号

**参考文章**：[首板二波形态详解](https://mp.weixin.qq.com/s/eJ-NMgMUo1ckknSvT8C_vA)

#### 强势二波

识别大涨后震荡整理、即将开启第二波的龙头股：

**识别条件**：
- **大涨段**：至少4个涨停 OR 10日内涨幅≥50%
- **震荡段**：5-30天横盘震荡，最大回撤≤20%，无涨停/跌停
- **MACD**：要求金叉

**输出状态**：
| 状态 | 条件 | 信号 |
|------|------|------|
| 震荡中 | 震荡≥5天，距前高≥3% | 持续观察 |
| 震荡末期 | 震荡≥10天，距前高<3% | 即将突破，可关注 |

#### 揉搓线

识别主力洗盘后的拉升信号：

**识别条件**：
- **第一根K线**：长上影线（上影线≥实体1.5倍）
- **第二根K线**：长下影线（下影线≥实体1.5倍）
- **缩量**：两日成交量相对前期萎缩
- **趋势**：处于上升趋势或震荡上行的个股效果更佳

**输出评分**：
- 评分 ≥75：高置信度，形态良好
- 评分 60-74：中等置信度，可关注
- 评分 <60：低置信度，谨慎观察

### 4. 形态回归分析

分析历史匹配的后续走势统计：

```bash
pichip analyze 002594 2024-01-01 2024-02-01
```

输出包括：
- 历史匹配次数
- 各时间窗口（3/5/10/20日）胜率
- 平均涨幅、最大涨幅、最大亏损
- 投资建议

### 8. 龙虎榜查询

查询龙虎榜数据和机构/游资动向：

```bash

pichip lhb

# 查看指定日期龙虎榜
pichip lhb --date 20260224

# 查看指定股票龙虎榜详情
pichip lhb --stock 603887

```

**龙虎榜数据来源**：东方财富

### 9. 热榜股票筛选

多维度热榜股票筛选，识别机构+游资合力标的：

```bash
# 扫描热榜股票（默认显示前20只）
pichip hot

# 指定日期
pichip hot --date 20260224

# 最低分数过滤
pichip hot --min-score 60

# 评级过滤（A/B/C/D）
pichip hot --grade A

# 返回前N只
pichip hot --top-n 50
```

**评分模型（100分制）**：

| 维度 | 分值 | 说明 |
|------|------|------|
| 多榜共振 | 40分 | 出现在涨幅榜、量比榜、换手率榜、连板榜等多个榜单 |
| 资金性质 | 30分 | 龙虎榜机构/游资买卖分析 |
| 技术形态 | 20分 | 均线位置、突破情况、量价配合 |
| 板块效应 | 10分 | 板块涨停数、板块涨幅 |

**评级标准**：
- **A级（80-100分）**：强烈关注，机构游资合力+多榜共振+板块龙头
- **B级（60-79分）**：重点关注，有2-3个亮点
- **C级（40-59分）**：一般关注，需进一步观察
- **D级（<40分）**：放弃，风险大于机会

**风险过滤（一票否决）**：
- ST/*ST股票
- 连续一字板
- 高位放巨量（换手率>40%）
- 买一席位占比>50%（疑似对倒）
- 板块当日跌幅>2%
- 股价<3元

### 10. 主力控盘指数

量化主力资金对股票的控制程度，判断是否有主力关注：

```bash
# 查询单只股票控盘指数
pichip control --code 000001

# 扫描全市场高控盘股票
pichip control --scan

# 指定最低得分
pichip control --scan --min-score 70

# 返回前N只
pichip control --scan --top-n 30
```

**评分模型（100分制）**：

| 指标 | 权重 | 说明 |
|------|------|------|
| 筹码集中度 | 30% | 股东户数变化分析，户数减少=筹码集中 |
| 换手率趋势 | 20% | 换手率递减=主力锁仓，递增=筹码分散 |
| 量价控盘系数 | 30% | 缩量上涨天数占比，主力锁仓拉升 |
| 抗跌性 | 20% | 大盘下跌日个股相对表现 |

**无股东数据时（自动调整）**：

| 指标 | 权重 |
|------|------|
| 换手率趋势 | 35% |
| 量价控盘系数 | 35% |
| 抗跌性 | 30% |

**控盘等级划分**：
- **高度控盘（80-100分）**：主力主导，可关注
- **中高控盘（60-79分）**：有主力关注
- **中度控盘（40-59分）**：主力存在但力度一般
- **低控盘（20-39分）**：主力参与度低
- **无控盘（<20分）**：散户主导

**数据来源与缓存**：
- 股东户数：优先本地缓存 → akshare（东方财富），缓存有效期30天
- 大盘指数：优先本地缓存 → akshare，缓存有效期1天
- 换手率/成交额：本地K线数据
- 无需 tushare 高级权限即可使用

**数据同步命令**：
```bash
# 同步股东户数数据到本地缓存（首次使用或季度更新后执行）
pichip control --sync-holder

# 同步大盘指数数据到本地缓存
pichip control --sync-index
```

### 11. 健康回踩扫描

扫描涨停/大阳后健康回踩的股票，寻找缩量回调后的买入机会：

```bash
# 健康回踩选股（默认参数）
pichip scan pullback

# 指定参数
pichip scan pullback --days-back 5 --max-pullback 5 --min-volume-shrink 0.65

# 严格模式：只看站稳MA5的
pichip scan pullback --strict

# 只选热门板块股票
pichip scan pullback --hot-sector --min-hot-score 30

# 包含ST股票
pichip scan pullback --include-st

# 最低评分过滤
pichip scan pullback --min-score 80 --top-n 20
```

**识别条件**：
- **涨停/大阳**：近N天内出现涨停或大阳（主板≥7%，创业板≥12%）
- **健康回踩**：涨停后回踩2-5天，幅度≤5%
- **缩量**：回踩期间成交量萎缩（默认萎缩≥35%）
- **站稳均线**：回踩期间站稳MA5（可选严格模式）

**评分模型（100分制）**：

| 维度 | 分值 | 说明 |
|------|------|------|
| 回踩幅度 | 30分 | 小幅回踩(-1%~+1%)得分最高 |
| 缩量程度 | 30分 | 缩量越多得分越高 |
| MA5支撑 | 20分 | 站稳MA5得分 |
| 涨停质量 | 20分 | 涨停日换手率适中得分高 |

### 12. 背离扫描

扫描全市场的MACD背离信号（顶背离/底背离）：

```bash
# 扫描全部背离信号
pichip divergence

# 只扫描底背离（买入信号）
pichip divergence --type bottom

# 只扫描顶背离（卖出信号）
pichip divergence --type top

# 指定日期范围
pichip divergence --days-back 30

# 最低评分过滤
pichip divergence --min-score 60 --top-n 30
```

**背离类型**：
- **底背离**：价格创新低，但MACD柱子没创新低 → 潜在买入信号
- **顶背离**：价格创新高，但MACD柱子没创新高 → 潜在卖出信号

**评分模型（100分制）**：

| 维度 | 分值 | 说明 |
|------|------|------|
| 背离强度 | 40分 | 价格与MACD背离程度越大得分越高 |
| 背离位置 | 30分 | 底背离在低位得分高，顶背离在高位得分高 |
| 确认信号 | 30分 | 是否有K线形态确认（如金叉/死叉） |

## 配置文件

配置文件 `config.yaml` 位于项目根目录：

```yaml
# 数据源配置
data_source:
  tushare_token: "your_tushare_token"  # tushare token（主数据源）
  # 获取方式：https://tushare.pro/register

# 调度器配置
scheduler:
  enabled: true
  noon_sync_time: "11:35"      # 中午同步时间（收盘后5分钟）
  sync_time: "16:05"           # 下午增量同步时间
  full_sync_time: "03:00"      # 全量同步时间（周日）
  sync_weekdays: [1, 2, 3, 4, 5]
  verify_time: "17:00"         # 验证时间

# 同步配置
sync:
  recent_days: 30
  full_years: 3

# 匹配配置
match:
  default_volume_weight: 0.0
  default_min_corr: 0.7

# LLM 配置（可选）
llm:
  enabled: false
  provider: "openai"
  model: "gpt-4o-mini"
  api_key: ""  # 或设置环境变量 OPENAI_API_KEY
```

### match 命令参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `stock` | 目标股票代码 | 必填 |
| `start` | 起始日期 YYYY-MM-DD | 必填 |
| `end` | 结束日期 YYYY-MM-DD | 必填 |
| `--top-n` | 返回结果数 | 20 |
| `--min-corr` | 最低相关系数 | 0.7 |
| `--latest` | 只匹配最新形态 | False |
| `--volume-weight` | 量能权重 | 0.0 |

### board 命令参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `pichip board list --type` | 板块类型: industry/concept/all | all |
| `pichip board sync --type` | 板块类型: industry/concept/all | all |
| `pichip board sync --start-date` | 开始日期 YYYYMMDD | 1年前 |
| `pichip board sync --end-date` | 结束日期 YYYYMMDD | 今天 |
| `pichip board show <name>` | 板块名称或代码 | 必填 |
| `pichip board show --days` | 显示最近N天数据 | 30 |

**示例：**
```bash
# 列出所有行业板块
pichip board list --type industry

# 同步板块K线数据（最近1年）
pichip board sync

# 同步指定时间范围的板块数据
pichip board sync --start-date 20240101 --end-date 20241231

# 显示半导体板块K线
pichip board show 半导体 --days 60
```
| `--board` | 限定行业板块 | - |
| `--concept` | 限定概念题材 | - |
| `--min-mv` | 最小市值（亿） | - |
| `--max-mv` | 最大市值（亿） | - |
| `--chart` | 生成对比图 | False |

## 数据存储

数据存储在 `data/pichip.db` SQLite 数据库中：

- `stock_daily`: K线数据（OHLCV + 换手率）
- `stock_info`: 股票基本信息（市值、换手率等）
- `match_records`: 匹配历史记录（含后续走势验证）

**数据范围**：A股主板、创业板、科创板，排除北交所股票。

## 注意事项

1. **数据源**：优先使用 tushare（需配置token），akshare 作为备用兜底
2. **数据同步时间**：首次同步约需 30-60 分钟，请耐心等待
3. **涨跌幅限制**：自动识别 ST股(5%)、创业板/科创板(20%)、主板(10%)
4. **北交所排除**：暂不支持北交所股票（代码以4/8开头）
5. **形态识别**：基于历史形态识别，不构成投资建议
6. **风险提示**：股市有风险，投资需谨慎
