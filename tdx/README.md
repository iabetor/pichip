# 通达信主力控盘指数公式

## 安装方法

1. 打开通达信软件
2. 按 `Ctrl+F` 打开公式管理器
3. 点击"新建" → 选择"技术指标" → "副图"
4. 将 `主力控盘指数.txt` 中的内容复制粘贴到编辑框
5. 公式名称填：`主力控盘指数`
6. 点击"确定"保存

## 指标说明

### 综合控盘指数

通过4个维度评估主力控盘程度：

| 子指标 | 权重 | 说明 |
|--------|------|------|
| 换手率趋势 | 20% | 换手率递减=吸筹锁定 |
| 量价控盘系数 | 30% | 缩量上涨=控盘强 |
| 抗跌性 | 25% | 大盘跌时抗跌=护盘 |
| 独立走势 | 25% | 低相关性=独立运作 |

### 控盘等级

- **高控盘** (≥80分): 主力高度控盘，筹码集中
- **中高控盘** (60-80分): 主力控盘较强
- **中度控盘** (40-60分): 有一定控盘
- **低控盘** (20-40分): 控盘较弱
- **无控盘** (<20分): 散户主导

### 信号说明

- 🟢 上箭头：控盘指数突破60，控盘增强
- 🔴 下箭头：控盘指数跌破60，控盘减弱

## 与Python版本的区别

| 功能 | Python版 | 通达信版 |
|------|----------|----------|
| 筹码集中度 | ✅ 有（股东户数） | ❌ 无（数据不可得） |
| 换手率趋势 | ✅ 有 | ✅ 有 |
| 量价控盘 | ✅ 有 | ✅ 有 |
| 抗跌性 | ✅ 有 | ✅ 有 |
| 独立走势 | ✅ 有 | ✅ 有（简化版） |
| 买入信号 | ✅ 缩量洗盘/平台突破等 | ❌ 仅控盘强弱信号 |

## 注意事项

1. 通达信版本无法获取股东户数数据，因此不包含"筹码集中度"指标
2. 相关系数使用"涨跌方向一致性"简化替代
3. 建议结合其他指标综合判断
