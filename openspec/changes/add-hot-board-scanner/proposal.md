# Change: Add Hot Board Stock Scanner

## Why

当前项目专注于K线形态识别，但缺乏对市场热度和资金流向的分析能力。热榜股票筛选是短线交易的重要工具，通过整合龙虎榜、涨幅榜、量比榜等多维度数据，可以帮助用户快速识别市场热点和机构/游资合力标的。

## What Changes

### 新增功能
- 新增 `pichip lhb` 命令：龙虎榜数据查询和分析
- 新增 `pichip hot` 命令：热榜股票综合筛选
- 支持多榜单数据获取：龙虎榜、涨幅榜、量比榜、换手率榜、连板榜
- 实现机构+游资合力榜分析
- 实现多维度评分模型（满分100分）

### 评分维度
1. **多榜共振（40分）**：股票出现在多个榜单中
2. **资金性质（30分）**：龙虎榜机构/游资买卖分析
3. **技术形态（20分）**：价格位置、突破情况、量价配合
4. **板块效应（10分）**：板块涨停数、板块涨幅

### 风险过滤（一票否决）
- ST/*ST股票
- 连续一字板
- 高位放巨量（换手率>40%）
- 买一席位占比>50%
- 板块当日跌幅>2%
- 股价<3元

### 输出格式
- 股票代码、名称、评分、评级（A/B/C/D）
- 出现的榜单列表
- 资金性质（机构/游资/混合）
- 板块及板块强度
- 核心亮点和风险提示
- 建议操作

## Impact

- Affected specs: 新增 `hot-board` 能力
- Affected code:
  - `pichip/cli.py` - 新增 `lhb` 和 `hot` 命令
  - `pichip/data/fetcher.py` - 新增龙虎榜数据获取函数
  - `pichip/hot_board/` - 新模块，包含评分引擎和筛选逻辑
- Dependencies: 使用现有 akshare 接口，无需新增依赖
