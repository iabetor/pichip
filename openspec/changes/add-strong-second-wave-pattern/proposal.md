# Change: Add "Strong Second Wave" Pattern Recognition

## Why

当前系统已有"首板二波"形态识别，但该形态要求"单涨停后缩量回调"，适用范围较窄。实际市场中存在更强势的形态：**一波大涨（可能有多个涨停）后震荡整理，再开启第二波上涨**。

典型案例如天奇股份（002009）：
- 2025-12-24~12-30：4连板暴涨
- 2025-12-31~2026-01-22：横盘震荡15天
- 2026-01-23起：开启第二波大涨

这种形态比"首板二波"更强势，能捕捉到龙头股的二波机会。

## What Changes

- **ADDED** 强势二波形态识别器 (`StrongSecondWavePattern`)
  - 识别大涨段（至少4个涨停 OR 10日内涨幅≥100%）
  - 识别震荡段（5-30天，最大回撤≤20%）
  - 判断当前状态：震荡中 / 震荡末期
- **ADDED** CLI 命令支持：`pichip pattern --type strong_second_wave`
- **ADDED** LLM 工具支持：自然语言查询"强势二波"

## Impact

- Affected specs: pattern
- Affected code:
  - `pichip/pattern/strong_second_wave.py` - 新增形态识别器
  - `pichip/pattern/__init__.py` - 注册新形态
  - `pichip/cli.py` - 新增命令参数
  - `pichip/llm/tools.py` - 新增工具支持

## Research Data

基于智光电气（002169）和天奇股份（002009）的实证分析：

| 指标 | 天奇股份 | 智光电气 |
|------|---------|---------|
| 大涨段涨停数 | 4连板 | 2涨停 |
| 大涨幅度 | ~50% | ~60% |
| 震荡天数 | 15天 | 19天 |
| 最大回撤 | 9.2% | 8.5% |
| 震荡期量能/大涨期量能 | 1.35（略放量） | 0.72（缩量） |

## Parameters

| 参数 | 实现值 | 说明 |
|------|--------|------|
| 大涨定义 | 至少4个涨停 OR 10日内涨幅≥50%（满足其一） | 强势大涨定义 |
| 震荡天数 | 5~30天 | 给足范围 |
| 最大回撤 | ≤20% | 两只样本都<10%，放宽到20% |
| 震荡末期判定 | 震荡≥10天 且 距前高<3% | 预判即将突破 |
| MACD要求 | 必须金叉 | 二波启动的必要条件 |
| 震荡期涨停 | 不允许 | 有涨停说明还在涨 |
| 震荡期跌停 | 不允许 | 有跌停说明走弱 |

## 参数调整历史

### 2026-02-23 参数优化

**问题**：初始参数过于严苛，导致无法筛选出任何结果。

**调整内容**：
- **大涨涨幅**：100% → 50%
  - 原因：天奇股份涨幅~50%，智光电气涨幅~60%，100%要求过高
  - 翻倍的股票在震荡期间风险较大，50%更符合实际案例

## Output States

| 状态 | 条件 | 信号 |
|------|------|------|
| 震荡中 | 震荡天数≥5天，距前高≥3% | 持续观察 |
| 震荡末期 | 震荡≥10天，距前高<3% | 即将突破，可关注 |
| 已突破 | 当前价≥前高×1.03 | 不再输出 |
